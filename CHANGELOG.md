# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-17

### Added
- Initial release with semantic versioning
- Vault naming functionality
- Theme configuration synced across devices
- Client-side file encryption with AES-GCM
- TOTP two-factor authentication
- Chunked file uploads for large files
- Folder support with nested structure
- Batch operations for files and folders
- Multiple theme options (Dark, Light, Ultra Dark, High Contrast)
- Image thumbnail generation with caching
- File preview for images
- Responsive file name truncation
- Storage quota management
- Session timeout configuration
- File search and filtering
- Grid and list view modes
- File renaming and moving
- Version display in UI

### Security
- End-to-end encryption for all files
- Password hashing with PBKDF2
- TOTP-based two-factor authentication
- Encrypted metadata storage option
- Secure session management
- Rate limiting on sensitive endpoints
